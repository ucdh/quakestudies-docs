DatatypeProperty

qsr




URI:
http://quakestudies.canterbury.ac.nz#mime_type




Label:
mime_type




Definition:
Indicates format according to the Internet Media Types vocabulary




Notes:
Mime types can be obtained from the Internet MIME Media Types vocabulary maintained by the Internet Assigned Numbers Authority: http://www.iana.org/assignments/media-types/index.html. http://mediatypes.appspot.com/ provides a means of associating a MIME type with a stable URI.

Functional

Part




    

  • An instance of the Part class
    • 

  • Represents the Part class, and as such is a container element. Part is used as a given Object may have two or more component parts which have different availability criteria, format specifications, and creator and contributors. Think of a book (Object) which may have an author (Creator), but may also contain photographs created by other individuals.

Indicates either weighting or ordinal position
